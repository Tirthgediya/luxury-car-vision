🚗 Luxury Car Vision

A modern and visually appealing luxury car showcase website built using HTML, CSS, and JavaScript. This project highlights premium vehicles with a focus on elegant UI design, smooth navigation, and an engaging user experience.

🌐 Live Demo

🔗 https://Tirthgediya.github.io/luxury-car-vision/

📌 Features
🚘 Home page with luxury branding and visuals
📖 About section with brand/story details
🏎️ Car showcase / listing section
📞 Contact page with form
🧭 Navigation bar for smooth browsing
🎨 Premium and clean UI design

🛠️ Technologies Used
HTML5
CSS3
JavaScript

🚀 Getting Started

To run this project locally:

Clone the repository
git clone https://github.com/Tirthgediya/luxury-car-vision.git

Navigate to the project folder
cd luxury-car-vision

Open index.html in your browser
